
# PrimeHomes – Luxury Real Estate Platform

**PrimeHomes** is a premium real estate website showcasing luxury apartments, villas, and investment properties. It is designed to provide a high-end user experience with elegant Dark × Gold styling, smooth animations, and fully functional navigation.

---

## **Features**

- Modern, luxurious landing page design
- Responsive layout (mobile, tablet, desktop)
- Hero section with cinematic background
- Interactive Property Cards with hover effects
- Fully functional navigation:
  - Home, About, Properties, Services, Contact
- Contact form with FormSubmit / Getform integration
- Floating WhatsApp contact button
- Smooth scrolling and parallax effects
- Dark × Gold premium color scheme
- Built with **React + Tailwind CSS**
- Ready for deployment on **Netlify** or **Vercel**

---

## **Installation**

1. Clone the repository:

```bash
git clone https://github.com/muzansiddig/PrimeHomes.git
````

2. Navigate into the project folder:

```bash
cd PrimeHomes
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

---

## **Build for Production**

```bash
npm run build
```

The `build` folder is ready to be deployed on **Netlify**, **Vercel**, or any static hosting service.
